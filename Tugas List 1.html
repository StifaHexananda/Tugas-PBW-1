HTML
Hyper Text Markup LanguageHyper (HTML) adalah bahasa markup yang digunakan untuk membuat dan menyusun struktur halaman web. Html menggunakan elemen-elemen seperti heading, paragraf, gambar, tautan, tabel, dan daftar untuk menampilkan konten di browser.
CSS
Cascading Style Sheet (CSS) adalah bahasa script yang digunakan untuk mengatur tampilan dan tata letak elemen dalam halaman web yang dibuat dengan HTML
JavaScript
JavaScript (JS) adalah bahasa pemograman yang digunakan untuk membuat halaman web lebih interaktif dan dinamis. Dengan JavaSript, kita bisa manambahkan animasi, validasi formulir, menangani event (klik, hover), dan banyak lagi.
